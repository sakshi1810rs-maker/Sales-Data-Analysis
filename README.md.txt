# Sales Data Analysis using Python and SQL

This is a beginner-level data analysis project.

## Tools Used
- Python (Pandas)
- SQL
- Microsoft Excel

## Project Structure
Sales-Data-Analysis
│
├── sales_data.csv
├── analysis.py
├── queries.sql
├── charts.xlsx   ← 2 perfect charts
├── README.md
└── requirements.txt

## Description
The project involves analyzing sales data to:
- Identify overall sales trends  
- Understand category-wise sales performance  
- Study monthly sales patterns

## Output
- Python analysis using Pandas
- SQL queries for data extraction
- Excel charts for visualization

## Note
- Python screenshots are not included in the project folder but can be available locally.  
- SQL query files are included but not executed.  
- Excel charts are included in the project folder.
